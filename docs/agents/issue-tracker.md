# Issue Tracker: Linear

Issues are tracked in Linear.

## Workflow

Use the `linear` MCP server tools (e.g., `list_issues`, `create_issue`, `get_issue`, `save_issue`, etc.) to read and write tickets. Always format Linear issue IDs as markdown links (e.g., `[JUS-283](https://linear.app/justaiit/issue/JUS-283)`). When opening a new Linear task or issue, always ensure it is related to the relevant Linear project; if the project is unknown or doesn't exist, ask the user before creating the issue.
